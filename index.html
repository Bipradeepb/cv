What is Web Development?

Web Development is the process or way of creating a website used as an interface for displaying information between the servers and the request computers. Web development has mainly two divisions-front end and back-end. Front-end development mainly involves HTML,CSS and Javascript.

Steps to learn web development:

  1. Learn how internet and computers work.
  2. Start to learn about the different languages one by one.This includes:
    • HTML
    • CSS
    • Javascript
    • NODE.JS
    • Bootstrap
    • Mongodb
    • React
  3. Pick one.Learn one and then move to the next.
  4. Once you have finished learning all the languages apply them real time by designing a project.
  5. Hurray! You are now an established developer.
